# CodeAlpha Data Redundancy Removal System

## Project Overview
Cloud-based Data Redundancy Removal System developed using Flask and SQLite.  
This project detects and prevents duplicate records to ensure database accuracy and integrity.

## Features
- Detects duplicate entries
- Prevents redundant data storage
- Simple and clean web interface
- Uses SQLite database for storage

## Tech Stack
- Python (Flask)
- SQLite
- HTML
- CSS

## How to Run
pip install -r requirements.txt  
python app.py
